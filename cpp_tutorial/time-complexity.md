# 시간 복잡도 정리

| 자료구조   | 참조    | 탐색    | 삽입    | 삭제    |
| ---------- | ------- | ------- | ------- | ------- |
| 배열       | O(1)    | O(n)    | O(n)    | O(n)    |
| 스택       | O(n)    | O(n)    | O(1)    | O(1)    |
| 큐         | O(n)    | O(n)    | O(1)    | O(1)    |
| 연결리스트 | O(n)    | O(n)    | O(1)    | O(1)    |
| 맵         | O(logN) | O(logN) | O(logN) | O(logN) |

스택과 큐 같은 경우에는 가장 앞에 있는 요소를 참조한다면 O(1)이지만,
그게 아니라면, 랜덤접근이 아닌 순차접근만 가능해서 O(n)이 걸린다.
